Hugo 搭建 github page

Dec 24, 2019
1 min read

安装 hugo

pacman -S hugo

生成站点

hugo new site plus7wist-hugo
cd plus7wist-hugo/

主要修改 config.toml。

baseURL = "https://plus7wist.github.io/"
languageCode = "zh-cn"
title = "plus7wist's blog"
theme = "contrast"

注意 baseURL 得以 https 开头，而且得以 / 结尾。记得添加主题代码：

[submodule "themes/contrast"]
        path = themes/contrast
        url = https://github.com/niklasbuschmann/contrast-hugo.git

写文章

生成文章：

hugo new posts/journal.md

写完文章：

hugo server # 开启本地博客服务
hugo # 在 public/ 生成站点

发布站点

主要把 public 发布在远端。

#! /usr/bin/env bash

set -e

hugo
cd public

git add .
git commit -m "hugo build $(date)"
git push origin master
