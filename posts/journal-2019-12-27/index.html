Go - 变长参数函数

func Min(min int, args ...int) int {
    for _, v := range args {
        if v < min {
            min = v
        }
    }
    return min
}

func main() {
    var values = []int{1, 2, 3, 4}
    fmt.Println(Min(10, values...))
}

定义的时候，... 加在类型前面，调用的时候，... 加在类型后面。
